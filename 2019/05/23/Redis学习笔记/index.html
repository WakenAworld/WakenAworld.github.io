    Redis 数据类型

    String(字符串)

    • 设置指定key的值SET key value
    • 获取指定key的值GET key
    • 将给定key的值设为value，并返回key的旧值GETSET key value
    • 获取一个或多个给定key的值MGET key1[key2..]
    • 将值关联到key，并将key的过期时间设为secondsSETEX key seconds value
    • 只有在key不存在时设置key的值SETNX key value
    • 返回key所储存的字符串值的长度STRLEN key
    • 删除使用DEL keyname
