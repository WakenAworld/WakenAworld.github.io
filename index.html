Git基础操作命令

  • git init 初始化仓库
  • git add 将更改添加到仓库
  • git commit将更改提交到仓库
    • git commit -m "message"将更改提交到仓库，并用message作说明
    • git commit -a -m "message"免去git add这一步直接提交到仓库
  • git status查看当前版本库的状态，哪个改了，哪个还没提交等
  • git log用来查看commit记录
